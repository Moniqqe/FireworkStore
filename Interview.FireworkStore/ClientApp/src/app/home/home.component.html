Welcome to your test single-page application firework store!!

Purpose of this excercise is to verify candidate skills and understanding basics of .net and C#.

This solution represents a basic CRUD application. It consist of standard REST API and simple Angular front-end client. Your job is to achieve several goals:

  • DISPLAY to user table with fireworks data which sits in file. Fireworks.csv will play a role of data source.
  • User must ba able to PLACE NEW ORDERS. Feel free to combine any fireworks and quantity as you like. Orders.csv will play a role of data source.
  • User must be able to DISPLAY his current placed orders. Orders.csv will play a role of data source.
  • User cannot be able to DELETE existing orders.
    • Prepare required API endpoins and business logic to meet above small acceptance criteria using known to you best object and programming principles. To simplyfy implementation you can use mocked user data from MockStaticUser.cs

Also as general goals do following:

  • Setup and run the code. Fix errors if requried
  • Fill missing methods to fit their purposes.
  • Do the code review! Refactor code if you feel that something isn't right or can be written better.
  • Write unit tests. Feel free to choose testing framework and write most tests as you can.
